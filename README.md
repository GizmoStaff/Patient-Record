# Patient-Record
Desktop application for keeping track of patient information along with the database.

Features:
- add patient
- delete patient
- search by blood group, 
- report -> list of all patients along with the statistics

### BUILT
- IDE: NetBeans 13
- Java
- Spring
- Maven
- MySQL 
- JasperSoft Studio
